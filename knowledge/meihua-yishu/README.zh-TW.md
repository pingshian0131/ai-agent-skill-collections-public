# 梅花易數 Meihua Yishu

[![License: CC BY-NC-SA 4.0](https://img.shields.io/badge/License-CC%20BY--NC--SA%204.0-lightgrey.svg)](https://creativecommons.org/licenses/by-nc-sa/4.0/)

**[English](README.md)** | **[繁體中文](README.zh-TW.md)**

**梅花易數**（Meihua Yishu / Plum Blossom Numerology / Plum Blossom I Ching）是相傳為宋代邵雍（邵康節）所創的中國傳統易學占卜術。此專案提供一個專業的梅花易數占卜系統，可作為 **Claude Code 的 Skill** 使用。

> **其他名稱：** 梅花易数（簡體）、Mei Hua Yi Shu、Plum Blossom Oracle、梅花心易

## 為什麼選擇梅花易數？

### 決策思維框架，而非算命

與一般對占卜的誤解不同，梅花易數是一套**結構化思考工具**，根植於三千年的中華哲學智慧：

| 常見誤解 | 實際情況 |
|----------|----------|
| 「預測未來」 | 分析當前條件與可能的發展軌跡 |
| 「迷信」 | 基於模式識別的系統化框架 |
| 「神秘玄學」 | 從多角度審視局勢的哲學方法 |

### 核心價值

🎯 **決策清晰** — 面對複雜選擇時，卦象結構迫使你考慮可能忽略的因素

🔄 **多角度思考** — 體用框架系統地審視自我與環境、行動與反應的關係

⏱️ **時機意識** — 應期系統促使你思考「何時」行動，而非只想「做什麼」

🌊 **變化哲學** — 基於易經「唯一不變的是變化」原則，幫助接受不確定性並靈活應對

💭 **反思工具** — 占卜過程創造內省空間，深入思考你的處境

### 適用對象

- **創業者** — 判斷策略時機與合作關係
- **個人** — 重大人生抉擇（職業、感情、遷居）
- **好奇心強者** — 探索東方哲學與象徵思維
- **開發者** — 將傳統智慧整合到 AI 應用

### 梅花易數的優勢

相較於其他易經占法：
- **更快速** — 無需銅錢或蓍草，可用時間、數字或觀察起卦
- **更直覺** — 以八卦（8個符號）為基礎，無需記憶全部六十四卦
- **更貼切** — 體用分析直接關聯你的具體情境
- **更實用** — 清晰的吉凶判定框架，提供可行的建議

> *「易不預測命運，而是揭示規律。理解規律，方能做出更好的選擇。」*

## 功能特色

### 起卦方式
- **時間起卦** — 以當前或指定時間起卦
- **數字起卦** — 用數字來起卦
- **聲音起卦** — 以聽到的聲音次數起卦
- **顏色起卦** — 根據所見顏色對應五行起卦
- **尺寸起卦** — 以物品尺寸數字起卦
- **方位起卦** — 根據來人或事物方位起卦

### 解卦功能
- **體用生克分析** — 解讀卦象中的體卦與用卦關係
- **通關化解** — 分析五行通關緩解克象的技巧
- **六十四卦解讀** — 詳細解釋卦辭與爻辭
- **動爻變卦推演** — 分析變卦的吉凶
- **卦氣旺衰** — 根據季節判斷體卦強弱
- **應期推斷** — 預測事情應驗的時間

### 專門占法
- **十八類分占** — 婚姻、疾病、求財、出行等專門占法
- **十應判斷** — 天時、地理、人事等外應分析
- **八卦萬物類象** — 提供現代與傳統的類象對照
- **測字占卜** — 拆字、相字、字占

### 智能輔助
- **照片分析** — 上傳照片，AI 分析環境外應
- **環境感知** — 描述周遭聲音、顏色、人物，增強斷卦

## 使用方式

**簡單方式：** 在 Claude Code 中貼上網址，然後說「請幫我安裝這個 skill」：
```
https://github.com/muyen/meihua-yishu
```

**手動方式：**
```bash
# 個人技能（適用於所有專案）
git clone https://github.com/muyen/meihua-yishu.git ~/.claude/skills/meihua-yishu

# 或針對特定專案
git clone https://github.com/muyen/meihua-yishu.git .claude/skills/meihua-yishu
```

安裝後，提及占卜相關詞彙如「占卦」、「起卦」、「梅花易數」，技能會自動啟用。

### ChatGPT 自訂 GPT

1. 前往 https://chatgpt.com/gpts/editor 建立新 GPT
2. 將 `SKILL.md` 全部內容複製到 Instructions 欄位（必須 < 8000 字元）
3. 將 `references/` 資料夾中的全部 10 個檔案上傳為 Knowledge
4. 設定對話開場白：「幫我起卦」、「占一卦」、「測字」

### Google Gemini Gems

1. 前往 https://gemini.google.com/gems 建立新 Gem
2. 將 `SKILL.md` 全部內容複製到 Instructions 欄位
3. 將 `references/` 資料夾中的全部 10 個檔案上傳為 Knowledge（Gemini 限制：10 個檔案）
4. 設定建議提示：「幫我起卦」、「占一卦」、「測字」

python scripts/meihua_calc.py num 6 8

# 以數字起卦（三個數字，第三個為動爻）

## 核心原理

梅花易數以**先天八卦數**起卦：

| 卦 | 數 | 五行 | 符號 |
|---|---|------|------|
| 乾 | 1 | 金 | ☰ |
| 兌 | 2 | 金 | ☱ |
| 離 | 3 | 火 | ☲ |
| 震 | 4 | 木 | ☳ |
| 巽 | 5 | 木 | ☴ |
| 坎 | 6 | 水 | ☵ |
| 艮 | 7 | 土 | ☶ |
| 坤 | 8 | 土 | ☷ |

### 體用論

- **體卦**：主體、自己、問卦者
- **用卦**：客體、事情、外在環境
- **互卦**：事情發展過程
- **變卦**：最終結果

### 吉凶判定

| 情況 | 吉凶 | 說明 |
|------|------|------|
| 用生體 | 大吉 | 有進益之喜，外力相助 |
| 體克用 | 吉 | 我制約對方，可成事 |
| 體用比和 | 吉 | 五行相同，和諧順利 |
| 用克體 | 凶 | 受制於人，不利 |
| 體生用 | 耗洩 | 付出多回報少 |

### 通關化解

當體用相克時，若互卦或變卦中有「通關」元素，可緩解凶象：

| 克關係 | 通關元素 |
|--------|----------|
| 金克木 | 水 |
| 木克土 | 火 |
| 土克水 | 金 |
| 水克火 | 木 |
| 火克金 | 土 |

## 占卜原則

1. **無疑不卜** — 沒有具體問題不占
2. **筮不過三** — 同一問題不重複占卜超過三次
3. **不動不占** — 無事不占
4. **依理推斷** — 卦象需結合實際情況解讀
