# 卦象關係深度指南：應、承乘、互卦、錯卦、綜卦、變卦
*基於周易384爻統計分析*
# 快速參考：六種關係的作用
 關係  定義  作用  何時看
 應  初↔四、二↔五、三↔上  有沒有外援  問合作、求助
 承乘  相鄰兩爻上下關係  上下關係順不順  問職場、輩分
 互卦  爻2-5組成的卦  內在本質（最重要！） 每次必看
 錯卦  陰陽全反  互補視角  看對立統一
 綜卦  上下顛倒  對方視角  問感情、談判
 變卦  動爻變化後的卦  發展趨勢  有動爻時
# 一、應（對應爻）— 看外援
# 對應關係
初爻 ←→ 四爻
二爻 ←→ 五爻  ← 最重要（君臣相應）
三爻 ←→ 上爻
# 判斷規則
- 一陰一陽 = 有應（有人幫）
- 兩陽或兩陰 = 無應（孤軍奮戰）
# 統計發現（重要！）
 應位  有應吉率  無應吉率  效果
 二↔五  68.8%  51.6%  +17.2%
 初↔四  46.9%  43.8%  +3.1%
 三↔上  34.4%  28.1%  +6.3%
# 占卜應用
- 問事業：看體卦與用卦的二五爻是否有應
- 二五有應：有貴人、資源到位、合作順利
- 二五無應：需自力更生、缺乏外援
# 二、承乘 — 看上下關係
# 定義
承 = 下爻承托上爻
乘 = 上爻騎乘下爻
# 四種組合
 類型  吉率  凶率  意義
 陽乘陰  43.8%  25.0%  剛統柔，最順
 陰承陽  35.0%  35.0%  柔順剛，平衡
 陽承陰  中等  中等  剛在柔下
 陰乘陽  22.5%  50.0%  柔凌剛，最危
# 化解「陰乘陽」
三者具二即可轉吉：得中 + 得尊位 + 合時義
# 占卜應用
- 看體卦動爻與相鄰爻的陰陽關係
- 陰乘陽出現時：提醒問卦者注意上下關係、權力結構
# 三、互卦 — 看內在本質（最重要！）
# 計算方法
下互卦 = 爻 2, 3, 4
上互卦 = 爻 3, 4, 5
# 統計發現
- 影響力係數：0.7315（其他關係只有 0.19 基線）
- 互卦是所有關係中影響力最強的！
# 時間意義
- 用卦 = 近期（當前狀態）
- 互卦 = 中期（發展過程、真正走向）
- 變卦 = 遠期（最終結果）
# 占卜應用
- 每次解卦必看互卦
- 本卦吉但互卦凶 = 表面好但有隱患
- 本卦凶但互卦吉 = 困難中有出路
- 互卦生體 = 過程中有助力
- 互卦克體 = 過程中有阻礙
# 四、錯卦 — 看互補面
# 定義
所有爻陰陽互換
# 統計發現
- 吉凶相反率：只有 28.1%
- 不是「對立」而是「互補」
# 常見錯卦對
 本卦  錯卦  關係
 乾  坤  創造 ↔ 承載
 泰  否  通泰 ↔ 閉塞
 既濟  未濟  完成 ↔ 未完
# 占卜應用
- 理解事物的另一面
- 看到現象的互補性質
# 五、綜卦 — 看對方視角
# 定義
整卦 180 度翻轉
# 爻位對應
本卦初爻 ↔ 綜卦上爻
本卦二爻 ↔ 綜卦五爻
本卦三爻 ↔ 綜卦四爻
# 統計發現
- 吉凶相近率：64.3%（最高相似度）
- 8卦自對稱：乾、坤、頤、大過、坎、離、中孚、小過
# 占卜應用
- 問感情/談判：把卦倒過來看對方視角
- 你在初爻 → 對方在上爻
- 理解對方處境後更容易找到共識
# 六、變卦 — 看發展趨勢
# 計算方法
動爻陰陽互換後形成新卦
# 384變卦統計發現
 指標  數值
 之卦趨勢向上 (↑)  38.5%
 之卦趨勢向下 (↓)  38.5%
 之卦趨勢持平 (→)  22.9%
# 陰爻 vs 陽爻變化（重要！）
 爻性  變好率  變差率  占卜建議
 陰爻動  41.1%  35.9%  變化較安全
 陽爻動  35.9%  41.1%  需謹慎評估
規律：陰爻變動後更容易變好；陽爻變動後更容易變差（陽極必反）
# 各爻位變化特性（重要！）
 爻位  凶→吉  吉→凶  占卜建議
 三爻  8例（最多）  6例  困境中有轉機，不必過慮
 初爻  6例  15例  起步者變動風險高
 二爻  3例  14例  得中者不宜妄動
 五爻  2例  11例  尊位者不宜妄動
規律：
- 三爻本「多凶」，但變化後最易轉好（窮則變，變則通）
- 二五爻本「多吉」，但變動後容易變差（守成者慎動）
# 臨卦是關鍵轉折點（重要！）
 情況  幅度  意義
 變成臨卦  +0.83  好兆頭，接近成功
 離開臨卦  -0.83  需警惕，離開好位置
臨卦特性：二陽在下、四陰在上，代表「貴人降臨」「接近成功」
# 重要案例
凶爻變吉（有出路）：
 本卦  動爻  之卦  趨勢
 歸妹  四爻  臨  +0.83
 節  五爻  臨  +0.83
 師  初爻  臨  +0.67
 恆  上爻  鼎  +0.67
 乾  三爻  履  +0.50
吉爻變凶（好景不長）：
 本卦  動爻  之卦  趨勢
 臨  五爻  節  -0.83
 臨  四爻  歸妹  -0.83
 臨  初爻  師  -0.67
 比  二爻  坎  -0.67
 鼎  上爻  恆  -0.67
# 占卜應用
- 陰爻動 → 較安全
- 陽爻動 → 需謹慎
- 三爻動且本凶 → 不必過慮，變後可能轉好
- 二五爻動且本吉 → 考慮是否真需要變動
- 變成臨卦 → 好兆頭
- 離開臨卦 → 需警惕
- 動爻凶但變卦好 = 窮則變，變則通
- 動爻吉但變卦差 = 居安思危
# 解卦優先順序
# 權重排序（經驗證）
1. 時義 (40%)     ← 卦的整體主題
2. 爻位 (25%)     ← 動爻在什麼位置
3. 得中 (15%)     ← 是否二或五
4. 應 (10%)       ← 有沒有外援
5. 承乘 (7%)      ← 上下關係順不順
6. 當位 (3%)      ← 陰陽是否得位
# 完整解卦公式
完整解讀 = 本卦（現狀）
         + 動爻（轉折點）
         + 變卦（結果）
         + 互卦（內在本質，最重要！）
         + 應爻（外援）
         + 綜卦（對方視角，選用）
# 與梅花易數體用的結合
# 體用 + 六種關係
 步驟  看什麼  目的
 1  體用生克  基本吉凶
 2  互卦生克  中期走向（最重要）
 3  變卦生克  最終結果
 4  二五應  有無外援
 5  承乘關係  上下是否順
 6  綜卦  對方視角（選用）
# 範例
問：這次合作會成功嗎？
1. 體用：體卦震木，用卦坎水，水生木 → 基本有利
2. 互卦：若互卦為離火，木生火（體生互）→ 過程中有付出
3. 變卦：若變卦為乾金，金克木（變克體）→ 最終有阻礙
4. 二五應：若有應 → 合作方會配合
5. 承乘：若動爻陰乘陽 → 可能有人不服
6. 綜卦：倒過來看 → 對方視角如何

通關化解
體用相克時，互卦或變卦有通關元素可緩解凶象。
通關=能同時與兩個相克元素產生生關係的第三元素
通關對照表
克關係|通關|原理|卦象
金克木|水|金生水水生木|乾兌→坎→震巽
木克土|火|木生火火生土|震巽→離→坤艮
土克水|金|土生金金生水|坤艮→乾兌→坎
水克火|木|水生木木生火|坎→震巽→離
火克金|土|火生土土生金|離→坤艮→乾兌
通關應用
1確定體用關係（克關係則繼續）
2查看互卦變卦是否有通關元素
3有通關凶象減輕，無則維持
通關歌訣
金木相爭水通關，木土相克火當先。土水交戰金為媒，水火不容木來援。火金相搏土調停，五行通關化凶煞。
